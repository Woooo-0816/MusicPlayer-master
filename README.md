**SweetMusicPlayer**是一款基于百度音乐API的在线音乐播放器。拥有播放本地、在线音乐以及mv等功能，采用 MVP + RxJava + Retrofit 架构的项目, 使用Java和Kotlin混合开发。
更多请看[博客](http://blog.csdn.net/column/details/sweetmusicplayer.html)。

## Features  
- 本地音乐按照歌曲，专辑，艺术家显示对应信息
- 在线搜索音乐并播放
- 智能匹配本地歌词,在线歌词
- 歌词调整播放进度
- 定时停止播放音乐
- 在线音乐播放MV
- 检索本地音乐功能
